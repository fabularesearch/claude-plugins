# Fabula Claude Code Plugins

A self-hosted Claude Code plugin marketplace, serving as a directory of plugins associated with our research tools.

## Setup

Before you can install a Fabula plugin, you'll need to register our marketplace:

* Boot up Claude Code: `claude`.
* Inside your Claude Code session, add our marketplace:  `/plugin marketplace add fabularesearch/claude-plugins`.

Now you'll be able to install a plugin using its command from in the table below.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [`stiri`](https://github.com/fabularesearch/stiri) | Rapid experiments in representation engineering (RepE). | `/plugin install stiri@fabularesearch` |
